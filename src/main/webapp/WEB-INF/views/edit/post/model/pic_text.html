經營理念

中彩面對e世紀印刷技術發展趨勢，將持續2005年針對印刷新知識、新技術、新設備、品質提昇、製程改善、新材料原料分析等主題範疇進行的研究與評估；目前中彩已完成內湖數位媒體中心以及數個印前中心設置、生產流程自動化、CTP設備與技術導入、FTP檔案傳輸、E1點對點專線設立、成立24小時內客訴回覆作業、印刷現場看板化管理等專案計劃，並成功落實於實際生產作業中，積極朝向中彩「快（交期快速）、準（品質精準）、好（服務完好）」之品質政策目標邁進；
